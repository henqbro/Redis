## Задание 1

Кеширование решает следующие проблемы:
	•	Медленный доступ к данным — повторные запросы к БД или API отвечают из кеша мгновенно
	•	Перегрузка БД — кеш принимает на себя однотипные запросы, снижая нагрузку на основное хранилище
	•	Пиковые нагрузки — в моменты всплесков трафика система не «падает», а отдаёт данные из кеша
	•	Повторные тяжёлые вычисления — результат считается один раз и сохраняется для дальнейших запросов
	•	Избыточный сетевой трафик — браузерное и HTTP-кеширование уменьшает передаваемый объём данных и ускоряет загрузку страниц
